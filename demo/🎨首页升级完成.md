## 💡 使用提示

### 如果你想调整内容

1. **修改平台介绍文字**
   - 编辑 `HomeView.vue` 第71-73行

2. **修改数据统计**
   - 编辑 `HomeView.vue` 第75-92行

3. **修改车型信息**
   - 编辑 `HomeView.vue` 第204-278行（motorcycleTypes数组）

4. **修改颜色方案**
   - 编辑 `HomeView.vue` 的 `<style>` 部分
   - 主要颜色变量在各个类的 `background` 和 `color` 属性中

### 如果你想添加更多车型

在 `motorcycleTypes` 数组中添加新对象：
```javascript
{
  name: '车型名称',
  description: '车型描述',
  icon: `<svg>...</svg>`, // SVG图标
  features: ['特点1', '特点2', '特点3']
}
```
